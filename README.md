# hubot-pugme

Gets you the most inspirational quotes from http://inspirobot.me/

## Installation

In hubot project repo, run:

`npm install hubot-inspires --save`

Then add **hubot-inspires** to your `external-scripts.json`:

```json
[
  "hubot-inspires"
]
```
